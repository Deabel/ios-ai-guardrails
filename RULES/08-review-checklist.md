# Review Checklist

在提交 AI 生成代码前，检查以下问题。

## 架构

- 是否遵循当前工程现有模式？
- UI 层是否保持足够轻？
- 是否引入了无意义的新层？

## Swift 风格

- 命名是否清晰？
- 是否存在过深嵌套？
- 是否有强制解包风险？

## 并发

- 是否有主线程阻塞？
- 是否有共享可变状态竞态？
- 是否滥用了 `Task.detached`？

## 数据与错误

- 错误是否被显式处理？
- 是否存在静默失败？
- 网络 / 存储逻辑是否放在正确层级？

## UI

- 是否考虑 loading / empty / error？
- 是否存在明显 accessibility 或 localization 风险？

## 测试

- 需要补测试吗？
- 是否说明了回归点？

## 交付

- 是否有 debug residue？
- 是否会影响调用方？
- 是否需要迁移说明？
