# Codex Project Root Example

Place `AGENTS.md` at the repository root.

Optional:
Append project-specific constraints under `Local Project Addendum`.

Example additions:

- Module prefix is `RH`
- Prefer UIKit in legacy health device modules
- Prefer SwiftUI in new feature-only screens
- Reuse shared APIClient and logging layer
- Chinese comments are acceptable for team-facing code
- Avoid introducing a second analytics abstraction
